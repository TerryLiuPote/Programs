# 项目介绍书

## 项目名称
FP-tree 构建与推荐系统

## 项目概述
本项目旨在实现一个基于 FP-tree（频繁模式树）的数据挖掘系统。系统包括以下主要功能：
1. **数据生成器**：随机生成交易数据并保存到文件中。
2. **FP-tree 构建器**：从交易数据中构建 FP-tree，并将树结构保存到文件。
3. **推荐系统**：基于用户输入的物品，利用 FP-tree 推荐最可能的下一个物品。

## 项目组成

### 1. 数据生成器
- **文件**：`generate_DB_in.cpp`
- **功能**：
  - 随机生成交易数据。
  - 用户可指定生成的交易数量。
  - 每笔交易包含随机数量的物品。
  - 输出文件：`DB.in`。
- **示例输出**：
  ```
  apple banana orange
  banana orange
  apple orange
  ```

### 2. FP-tree 构建器
- **文件**：`PFP-tree.cpp`
- **功能**：
  - 从 `DB.in` 文件读取交易数据。
  - 构建 FP-tree 数据结构。
  - 将 FP-tree 的结构保存到 `Tree.out` 文件。
  - 输出物品的频率统计。
- **示例输出**：
  ```
  FP-tree:
  null (1)
  |-apple (3)
  | |-banana (2)
  | |-orange (1)
  ```

### 3. 推荐系统
- **文件**：`recommend_item.cpp`
- **功能**：
  - 从 `Tree.out` 文件加载 FP-tree。
  - 接受用户输入的物品序列。
  - 基于 FP-tree 推荐最可能的下一个物品。
- **示例交互**：
  ```
  Enter items (space-separated): apple
  Recommended item: banana
  ```

## 技术细节
- **编程语言**：C++
- **数据结构**：
  - 使用 `map` 和 `shared_ptr` 实现 FP-tree。
  - 使用 `unordered_map` 存储物品频率和节点映射。
- **文件操作**：
  - 使用 `ifstream` 和 `ofstream` 进行文件读写。

## 使用说明

### 1. 生成交易数据
运行 `generate_DB_in.cpp`，输入生成的交易数量。
```bash
./generate_DB_in.exe
```
生成的交易数据将保存在 `DB.in` 文件中。

### 2. 构建 FP-tree
运行 `PFP-tree.cpp`，程序将读取 `DB.in` 文件并构建 FP-tree。
```bash
./PFP-tree.exe
```
FP-tree 的结构将保存在 `Tree.out` 文件中。

### 3. 推荐物品
运行 `recommend_item.cpp`，输入物品序列，程序将推荐最可能的下一个物品。
```bash
./recommend_item.exe
```

## 项目亮点
- **模块化设计**：每个功能模块独立实现，便于维护和扩展。
- **高效算法**：利用 FP-tree 高效存储和挖掘频繁模式。
- **用户友好**：提供清晰的交互界面和调试信息。
